## 1. Resumen del proyecto

Por medio del presente proyecto se propone una pagina web la cual permite visualizar
un set de datos especificos de Studio Ghibli los cuales le permites al usuario descubri
informacion requerida, es por esto que por medio del desarrollo de este proyecto se 
permitira evidenciar la data por medio de filtros, ordenados y ciertos calculos.

Es por esto que se decide realizar una investigacion por medio de un formulario tipo 
encuesta el cual permitio identificar las sugenrencias y solicitudes de los usuarios 
brindando mas facilidad para el desarrollo de la pagina. 

![encuesta](https://github.com/Viviana8913/DEV007-data-lovers/assets/129759642/339888f9-d17b-4579-9603-4959fc4938a2)

Por medio de la encuesta e investigacion se logro acceder a las historias de usuario donde 
identificaron las solicitudes y requisitos de cada usuario brindando asi un mejor analisis de 
prioridad en la creacion de la pagina web 'Ghiblioteca'

![Historias de usuario](https://github.com/Viviana8913/DEV007-data-lovers/assets/129759642/8cd87ca1-9374-4003-8366-2a31be4235b1)

De esta forma se comienza con la creacion de los prototipos de diseño del sitio web el cual 
tuviera un enfoque en la resolucion de los problemas o solicitudes interpuestos por los usuarios. 

## 2. Diseño de la interfaz de usuario

Una vez realizada la investigacion por medio de la encuesta se procede a realizar los prototipos los 
acercan a una etapa mas real del proyecto deseado.

### Prototipo de baja fidelidad 

Este prototipo fue diseñado con el fin de brindare una guia del como y por que se brindarian los diferentes
accesos a la pagina web.

![prototipo de baja](https://github.com/Viviana8913/DEV007-data-lovers/assets/129759642/1f41e1ac-8e9a-4ac1-94af-8c45eab78ab4)

### Prototipo de alta fidelidad 

Con este prototipo de busca llevar a la realidad la queta exacta de lo que sera el sitio web y sus finalidades 
como la ubicacion de cada uno de sus funcionalidades, lo cual busca permitir al usuario la inteccion con la 
interfaz.

![Prototipo de alta](https://github.com/Viviana8913/DEV007-data-lovers/assets/129759642/b0d678e6-63ae-48b3-9bfe-fc5c60703cb0)

### Testeos 

Realizando los testeos se presentaron algunos errores por espacios o por constantes no determinadas pero se presentaron
testeos exitosos que permitieron llevar a cabo con gran satisfaccion la pagina web.
